# Baixando e configurando

- Abra um terminal:

    `Control"` + `Alt` + `t`

- Clone o repositório:

```bash
https://github.com/JoseGuilherme03/read_book.git
```

- Entre na pasta:
  
```bash
cd read_book
```

- Instale as dependências:

```bash
poetry install
```

- Ative o ambiente virtual:

```bash
poetry shell
```

- Entre na pasta do projeto:

```bash
cd read_book
``` 

- Execute o script:

```bash
python frequencia_palavra.py
```

- Se preferir, abra o vscode:

```bash
code .
```
